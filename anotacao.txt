Claro, aqui está um exemplo de como podemos usar PHP e MySQL para calcular a hora de término de um evento:

<?php

// Obter a hora de início do evento
$hora_de_inicio = date("H:i", strtotime("23:00"));

// Obter a duração do evento
$duracao = 2;

// Adicionar a duração à hora de início
$hora_de_termino = strtotime("+$duracao hours", strtotime($hora_de_inicio));

// Converter a data e hora resultantes para um formato legível por humanos
$hora_de_termino_formatada = date("H:i", $hora_de_termino);

// Imprimir a hora de término formatada
echo $hora_de_termino_formatada;

?>
Isso vai imprimir o seguinte:

01:00
